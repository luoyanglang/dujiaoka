# 项目介绍

## 什么是独角数卡？

独角数卡是一个开源的自动化数字商品销售系统，专为站长和个人开发者设计。它可以帮助你快速搭建一个功能完善的数字商品售卖平台。

## 核心特性

### 🎯 自动化销售
- 自动发货：购买后立即发送卡密
- 自动处理：无需人工干预
- 24/7 运营：全天候自动运行

### 💳 多支付支持
- 支付宝（当面付、PC、手机）
- 微信支付（扫码、企业）
- PayPal、Stripe
- 加密货币（USDT、TRX、ETH 等）
- 易支付、码支付等第三方

### 🛡️ 安全可靠
- 订单查询限流保护
- 支付请求日志记录
- XSS 攻击防护
- 输入验证增强
- 安全日志监控

### 🎨 灵活定制
- 多套前端模板
- 自定义模板开发
- 多语言支持
- 响应式设计

### 📊 管理便捷
- 可视化后台管理
- 商品分组管理
- 优惠券系统
- 订单统计分析
- 邮件通知

## 适用场景

### ✅ 适合销售
- 软件激活码
- 游戏点卡
- 视频会员
- 网盘账号
- 虚拟商品
- 数字内容

### ❌ 不适合销售
- 实物商品（无物流系统）
- 需要复杂售后的商品
- 违法违规商品

## 技术架构

### 后端技术
- **框架**: Laravel 6.x
- **语言**: PHP 7.4 / 8.0+
- **数据库**: MySQL 5.6+ / MariaDB
- **缓存**: Redis / File
- **队列**: Redis / Sync

### 前端技术
- **UI 框架**: Bootstrap 4
- **模板引擎**: Blade
- **响应式**: 支持移动端

### 后台管理
- **管理面板**: Dcat Admin
- **权限管理**: RBAC
- **数据导出**: Excel
